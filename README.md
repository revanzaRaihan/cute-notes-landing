# Cute Notes Landing Page 🌸✨

Welcome to the official landing page code for **Cute Notes 🌸 — Pastel Desktop Note Pad**. 

This is a beautiful, interactive product landing page designed to showcase the Cute Notes application, its features, screenshots, and setup guides. It features a live interactive notebook customizer allowing users to preview pastel colors, washi tapes, and stickers before downloading the app!

---

## 🎨 Features of the Landing Page
* **🌸 Interactive Note Customizer**: Test-drive the look and feel of note cards by switching between custom pastel backdrops, washi tapes, and cute stickers live in the browser.
* **📸 Live Screenshot Gallery**: Tabbed walkthrough demonstrating the dashboard, custom deletion warnings, and success toast alerts.
* **💻 Command Copy Utility**: Fast copy-paste snippets for standard setup and packaging commands.
* **✨ Pastel Aesthetics**: Built with micro-animations, glassmorphic card designs, custom gradients, and responsive layouts.

---

## 🛠️ Stack & Technologies
* **Framework**: React + Vite
* **Styling**: Vanilla CSS (using variables, custom keyframes, and scrollbars)
* **Icons**: [Lucide React](https://lucide-react.github.io/)

---

## 🚀 Getting Started

To run the landing page locally:

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Dev Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## 🌸 About the Cute Notes App
**Cute Notes** is a desktop notes application built with React, Vite, and Electron. Designed with a playful spiral-notebook aesthetic, it allows users to write, organize, and decorate notes with:
* **Curated Pastel Colors** (Pink, Lavender, Peach, Mint, Yellow)
* **Washi Tapes & Stickers** (stripes, polka dots, grids, hearts, cats, stars)
* **iOS-style Drag & Drop Folders**
* **Lined Spiral Notebook Editor** with auto-indenting lists
* **Native Windows/Desktop packaging** via Electron

> [!NOTE]
> For the core desktop application code, please refer to the main repository (or folder) containing the Electron setup.
